# Fenland Running Club website

A modern static website for Fenland Running Club, built with Astro, TypeScript and maintainable CSS.

## Commands

```bash
npm run dev
npm run build
npm run preview
```

## Content editing

- News posts: `src/content/news/`
- Events and calendar entries: `src/content/events/`
- Welfare and policy entries: `src/content/policies/`
- Committee, coaches and run leaders: `src/data/people.ts`
- FAQs: `src/data/faqs.ts`

Use `TODO` placeholders where club information needs confirmation. Do not store private member data, secrets or API keys in the repository.
