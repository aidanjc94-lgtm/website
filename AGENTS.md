# Project working rules

- This is the Fenland Running Club public website.
- Use British English throughout.
- Keep the club colours: purple `#4E056E` and red `#D90205`.
- Do not alter the club logo.
- Do not copy code, images, animations or wording from the Process Academy website. Use it only as high-level design inspiration.
- Build with Astro, TypeScript and simple maintainable CSS.
- Prefer static content files for news, events, policies and committee details.
- Keep the site mobile-first and accessible.
- All normal body text must meet WCAG AA contrast expectations.
- Do not add paid services, external APIs, analytics or tracking without being asked.
- Do not store secrets, API keys, passwords or private member data in the repository.
- Use placeholders marked TODO where club information needs confirmation.
- After code changes, run the relevant build command and report whether it passed.
- The project is done only when `npm run build` passes and the site has no obvious broken links in the pages created.
